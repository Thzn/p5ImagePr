Rotation

Simple method that implements image rotation. The method uses heavy floating point arithmetics on each pixels. Since there can be rounding errors when using floating point arithmetics the images can introduce noise patterns. To reduce noise smoothing methods can be used. Median filter is the best method can be used to reduce this types of salt-pepper noise. Since there is a noise pattern in the image frequency domain image processing can be help full.

rotate(image,x,y,deg)
image : Gray image
x,y : Coordinates of rotation origin
deg : Angle of rotation

var img2;
function preload(){
img2=loadImage("coin.jpg");
}
function setup() {

createCanvas(img2.width*2,img2.height);
var newim=rgbToGray(img2);
displayGray(rotate(newim,newim.width/2,newim.height/2,-45));
image(img2,img2.width,0);
}
